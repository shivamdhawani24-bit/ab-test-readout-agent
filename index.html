STATISTICAL CHECKS PERFORMED

CHECK

WHAT IT EVALUATES

TYPE

Significance Test

Two-proportion z-test — p-value, z-score, and confidence level at 90 / 95 / 99%

Lift Calculation

Absolute lift in percentage points and relative lift percentage between control and treatment

Sample Size

Minimum detectable effect validation — flags underpowered experiments before drawing conclusions

Guardrail Check

Secondary metric regression detection — agent will not recommend shipping if guardrail is failing

Novelty Effect

Flags experiments under 7 days — results may not reflect steady-state user behavior

Business Impact

Annualized revenue estimate from activation lift across the eligible user base
